# FlowPrint Support

FlowPrint is an early-stage, open-source developer tool. Support is provided on a best-effort basis.

## Get help

- Documentation: [README.md](README.md)
- Bug reports and feature requests: [GitHub Issues](https://github.com/tsirou694-prog/flowprint/issues)
- Security or privacy concern: do not post secrets or sensitive personal data in a public issue. Open a minimal issue requesting a private contact channel.

## Include in a bug report

- ChatGPT or Codex surface used;
- operating system when relevant;
- FlowPrint plugin version;
- expected behavior and actual behavior;
- the smallest reproducible prompt or fixture;
- validation exit code and sanitized error text.

Remove tokens, cookies, authentication files, personal data, unrelated file paths, and complete private conversations before submitting a report.

## Current support boundary

The documented macOS and Windows Codex CLI samples and the repository test suite are engineering evidence, not a promise of compatibility with every host version or workflow. Generated drafts remain subject to user review.
