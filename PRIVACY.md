# FlowPrint Privacy Policy

Effective date: July 21, 2026

FlowPrint is a skills-only plugin that helps users turn completed AI work into a reusable Agent Skill draft. It does not operate a publisher-controlled server, create a FlowPrint account, or send data to the publisher.

## Data FlowPrint may process

When a user invokes FlowPrint, the host ChatGPT or Codex environment may provide:

- the conversation visible in the current chat;
- files the user explicitly uploads or makes available in the current project;
- files inside a project directory that passes FlowPrint's evidence-scope checks;
- FlowPrint classification, validation, manifest, and draft files created at the user's request.

FlowPrint is designed not to reconstruct missing evidence from personal memory stores, shell history, global configuration, unrelated repositories, broad personal folders, authentication records, or other plugins.

## Storage and retention

FlowPrint does not intentionally transmit or retain user content on infrastructure operated by the publisher. When the user explicitly asks to save, validate, compile, or revise a draft, FlowPrint may create files in the active ChatGPT or Codex workspace. The user controls those workspace files and may delete them through the host environment.

OpenAI and any user-connected services process data under their own terms and privacy policies. FlowPrint does not control the host platform's logging, retention, or security practices.

## External actions

Classifying or compiling a Skill draft does not authorize installation, publishing, deployment, messaging, payment, deletion, or another external action. FlowPrint requires separate user authorization before an eligible host performs such an action.

## Personal information and secrets

Users should not provide passwords, access tokens, authentication files, or unrelated personal information. If a user makes sensitive information available to the host environment, FlowPrint attempts to keep evidence access within the current task and validated workspace scope, but no software can guarantee that user-provided content is risk-free.

## Children

FlowPrint is a developer workflow tool and is not directed to children.

## Changes

Material changes to this policy will be published in this repository with an updated effective date.

## Contact

For privacy questions, open a private support request through the contact method listed in [SUPPORT.md](SUPPORT.md). Do not include secrets or sensitive personal data in a public GitHub issue.
